Paramount Cookbook

The Paramount Cookbook is meant to be a collection of Chef Cookbooks that work together to configure a series of online services.

Stack Components

Cloud

djbdns::cache, owncloud, prosody, znc

Email

dovecot, postfix, roundcube, postgresql

Security

rkhunter,aide,fail2ban,firewall,ldap,openvpn,encfs`

System

ntp, sudo, sysctl, rsyslog

Web

nginx
